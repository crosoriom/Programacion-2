Instrucciones

Objetivo del Juego

El objetivo es conseguir la mayor puntuación posible según el modo de juego seleccionado.

Modos de Juego

  • Clásico: Juega sin límite de tiempo hasta completar todos los niveles.
  • Por Tiempo: Consigue la mayor puntuación posible en 3 minutos.
  • Zen: Modo relajado sin presión, ideal para principiantes.

Controles

  • Utiliza las teclas de dirección para moverte.
  • Presiona Espacio para acciones especiales.
  • Presiona P para pausar el juego.

Consejos

Practica primero en el modo Zen para familiarizarte con los controles antes de intentar los otros modos.
